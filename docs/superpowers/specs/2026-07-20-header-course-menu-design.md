# Динамічне підменю курсів у шапці

## Мета

Пункт «Курси» у мобільній шапці розкриває актуальні опубліковані курси.

## Рішення

- Дані беруться з CPT `course` зі статусом `publish`.
- Курси групуються за таксономією `course_direction`: `programming` і `english`.
- Курси без одного з цих напрямів не показуються.
- Кожен пункт веде на permalink курсу; списки не дублюються в ACF або у шаблоні.

## Перевірка

Runtime-тест перевіряє обидві групи, посилання на опубліковані курси та відсутність курсів без напряму. Browser smoke підтверджує розкриття пункту «Курси» на мобільному екрані.
