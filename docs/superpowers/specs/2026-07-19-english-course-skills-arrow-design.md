# Стрілка між блоками навичок на сторінках англійських курсів

## Мета

Додати на desktop декоративну сіру пунктирну стрілку, яка візуально проводить користувача від першої картки навичок до останньої в секції `.english-course-skills`.

## Рішення

- Використати наданий PNG-asset зі стрілкою.
- Підключити asset через псевдоелемент секції в існуючому `assets/css/english-course.css`.
- Розмістити стрілку позаду заголовка та карток, щоб вона виглядала як зв'язок між блоками.
- Додати `pointer-events: none`, щоб декоративний шар не блокував взаємодію.
- На вузьких екранах приховати стрілку: картки переходять у вертикальну структуру, де горизонтальна траєкторія втрачає сенс.
- Не змінювати PHP, ACF, JavaScript або залежності.

## Критерії приймання

1. На desktop стрілка має бути сірою, помітною та проходити від першої до останньої картки.
2. Картки та їхній текст залишаються поверх стрілки й повністю клікабельні.
3. На mobile стрілка не створює обрізання, горизонтальний скрол або перекриття контенту.
4. Сторінки англійських курсів продовжують використовувати існуючий CSS-файл без нової логіки рендерингу.
